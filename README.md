# Análisis de Nicho: Nu-Metal & Alt-Rock en Spotify 

Este proyecto utiliza Ciencia de Datos para explorar la popularidad y las características sonoras (energía, popularidad) de las bandas más icónicas de la escena Nu-Metal.

## Ver el Análisis Interactivo
Puedes ver el notebook con los gráficos y conclusiones directamente en Kaggle:
[Link al Proyecto en Kaggle](https://www.kaggle.com/code/claudiotones/analisis-de-nicho-nu-metal-90)

## Herramientas Utilizadas
* **Lenguaje:** Python
* **Librerías:** Pandas, Seaborn, Matplotlib, Kagglehub
* **Dataset:** Spotify Tracks Dataset (Kaggle)

## Hallazgos Clave
* Linkin Park lidera en volumen de hits y popularidad constante.
* Bandas como Deftones logran alta popularidad incluso en tracks de menor energía (atmosféricos).
* Existe una correlación directa entre la energía y los éxitos más comerciales de Limp Bizkit.
